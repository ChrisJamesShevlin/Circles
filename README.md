# 🌿 First Generative Art Experiment

This is my first attempt at creating generative art using Python and py5.

I started with a very simple idea: drawing circles that follow the mouse. But as the sketch keeps running, the repeated motion builds up into these dense, organic patterns that almost look alive.

## 🧠 How it works (simple idea)

The sketch continuously draws circles at the mouse position:

* Every frame, a new circle is drawn
* Nothing gets cleared from the screen
* Over time, layers build up into complex shapes

It’s basically:

> simple rules + repetition = unexpected complexity

## 🎨 What I like about it

* The patterns feel almost like tangled vines or flowing energy
* You don’t fully control the result — it emerges over time
* Small movements create big visual differences

## 💾 Interaction

* Move the mouse → draw
* Press **Enter** → capture the current image

## 🌱 What I learned

* How real-time drawing works (frame-by-frame)
* How small pieces of code can create complex visuals
* How to use input (mouse + keyboard) to control output
* That generative art is more about systems than single drawings

## 🚧 What I want to explore next

* Using the Fibonacci sequence to control size and spacing
* Creating spirals instead of free movement
* Adding variation (color, randomness, motion rules)
